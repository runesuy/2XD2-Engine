enum class e2XD::framework::ProcessMode
strong
Enumerator
DEFAULT 

Use the node's standard processing behavior: inherit pause state and process mode from the parent unless explicitly overridden. The node reacts normally to pause changes in its parent.

ALWAYS_INHERIT 

Always inherit the process mode from the parent node.

STUBBORN 

Does not react to pause changes in parent nodes.

ALWAYS_RUNNING 

Always call update on this node, regardless of its visibility or state.

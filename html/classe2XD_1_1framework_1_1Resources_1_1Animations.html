Load animations in the manager using a json config file. Json format:

{
"animation-map-name": "example-animations",
"animations": [
{
"name": "animation-1",
"frame-duration": 0.1,
"repeat": false,
"frames": [
"texture_0",
"texture_1",
"texture_2,"
]
}
]
}
The mentioned textures need to be textures present in SFMLTextureManager

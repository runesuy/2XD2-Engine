Contributing

Hey, thank you for considering contributing to 2XD2-Engine! Your contributions are highly appreciated and help improve the engine for everyone.

How to Contribute

1. Fork the Repository

Start by forking the 2XD2-Engine repository to your own GitHub account.

2. Create a New Branch

Create a new branch for your feature or bug fix. Use a descriptive name for your branch, such as feature-new-node or bugfix-event-handling.

3. Make Your Changes

Make the necessary changes in your branch. Ensure that your code follows the existing coding style and conventions used in the project. Please also add appropriate tests for your changes to ensure they work as expected.

4. Commit Your Changes

Commit your changes with clear and concise commit messages that describe the purpose of the changes.

5. Push to Your Fork

Push your changes to your forked repository on GitHub.

6. Create a Pull Request

Go to the original 2XD2-Engine repository and create a pull request from your forked branch to the main repository. Provide a detailed description of your changes and why they should be merged

Someone will review your pull request and may provide feedback or request changes before merging.

All tests should pass before submitting a pull request.

Discord

Join our Discord server to discuss ideas, ask questions, and get help from the community. Contributors can get a special role that gives them access to the development channels.

We look forward to your contributions and thank you for helping to make 2XD2-Engine better!

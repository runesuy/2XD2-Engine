\doxysubsection{Detailed Description}
Input\+Handler interface to manage input from keyboard, mouse, and other devices Can be provided to the framework \doxylink{classe2XD_1_1framework_1_1Game}{Game} through \doxylink{classe2XD_1_1framework_1_1IGameConfig}{IGame\+Config}. When provided to the framework \doxylink{classe2XD_1_1framework_1_1Game}{Game}, it will be used by the static \doxylink{classe2XD_1_1framework_1_1Input}{Input} class to provide input states throughout the engine.

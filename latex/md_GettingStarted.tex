Before you begin, ensure you have the following installed on your system\+:
\begin{DoxyItemize}
\item A C++ compiler that supports C++11 or later.
\item CMake (version 3.\+10 or later).
\item SFML library (version 2.\+5 or later). You can download it from the \href{https://www.sfml-dev.org/download.php}{\texttt{ official SFML website}}.
\item Git (optional, for cloning the repository).
\item A code editor or IDE of your choice (e.\+g., Visual Studio, CLion, VSCode).
\item Basic knowledge of C++ and CMake.
\end{DoxyItemize}\hypertarget{md_GettingStarted_autotoc_md10}{}\doxysection{\texorpdfstring{Installation}{Installation}}\label{md_GettingStarted_autotoc_md10}
\hypertarget{md_GettingStarted_autotoc_md11}{}\doxysubsection{\texorpdfstring{Using Cmake}{Using Cmake}}\label{md_GettingStarted_autotoc_md11}

\begin{DoxyEnumerate}
\item Download or clone the 2XD2-\/\+Engine repository\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{git\ clone\ https://github.com/runesuy/2XD2-\/Engine.git}

\end{DoxyCode}

\item In your CMake\+Lists.\+txt, add the following lines to include 2XD2-\/\+Engine as a subdirectory\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{add\_subdirectory(path/to/2XD2-\/Engine)}
\DoxyCodeLine{target\_link\_libraries(your\_project\_name\ PRIVATE\ 2XD2-\/Engine)}

\end{DoxyCode}

\item The library will automatically link against the SFML libraries. Ensure that SFML is correctly installed and accessible in your development environment.

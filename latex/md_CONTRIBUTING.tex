 Hey, thank you for considering contributing to 2XD2-\/\+Engine! Your contributions are highly appreciated and help improve the engine for everyone.\hypertarget{md_CONTRIBUTING_autotoc_md14}{}\doxysection{\texorpdfstring{How to Contribute}{How to Contribute}}\label{md_CONTRIBUTING_autotoc_md14}
\hypertarget{md_CONTRIBUTING_autotoc_md15}{}\doxysubsection{\texorpdfstring{1. Fork the Repository}{1. Fork the Repository}}\label{md_CONTRIBUTING_autotoc_md15}
Start by forking the 2XD2-\/\+Engine repository to your own Git\+Hub account. \hypertarget{md_CONTRIBUTING_autotoc_md16}{}\doxysubsection{\texorpdfstring{2. Create a New Branch}{2. Create a New Branch}}\label{md_CONTRIBUTING_autotoc_md16}
Create a new branch for your feature or bug fix. Use a descriptive name for your branch, such as {\ttfamily feature-\/new-\/node} or {\ttfamily bugfix-\/event-\/handling}. \hypertarget{md_CONTRIBUTING_autotoc_md17}{}\doxysubsection{\texorpdfstring{3. Make Your Changes}{3. Make Your Changes}}\label{md_CONTRIBUTING_autotoc_md17}
Make the necessary changes in your branch. Ensure that your code follows the existing coding style and conventions used in the project. Please also add appropriate tests for your changes to ensure they work as expected. \hypertarget{md_CONTRIBUTING_autotoc_md18}{}\doxysubsection{\texorpdfstring{4. Commit Your Changes}{4. Commit Your Changes}}\label{md_CONTRIBUTING_autotoc_md18}
Commit your changes with clear and concise commit messages that describe the purpose of the changes. \hypertarget{md_CONTRIBUTING_autotoc_md19}{}\doxysubsection{\texorpdfstring{5. Push to Your Fork}{5. Push to Your Fork}}\label{md_CONTRIBUTING_autotoc_md19}
Push your changes to your forked repository on Git\+Hub. \hypertarget{md_CONTRIBUTING_autotoc_md20}{}\doxysubsection{\texorpdfstring{6. Create a Pull Request}{6. Create a Pull Request}}\label{md_CONTRIBUTING_autotoc_md20}
Go to the original 2XD2-\/\+Engine repository and create a pull request from your forked branch to the main repository. Provide a detailed description of your changes and why they should be merged

Someone will review your pull request and may provide feedback or request changes before merging.

All tests should pass before submitting a pull request.\hypertarget{md_CONTRIBUTING_autotoc_md21}{}\doxysection{\texorpdfstring{Discord}{Discord}}\label{md_CONTRIBUTING_autotoc_md21}
Join our \href{https://discord.gg/vzNqgxz6W2}{\texttt{ Discord server}} to discuss ideas, ask questions, and get help from the community. Contributors can get a special role that gives them access to the development channels.

{\bfseries{We look forward to your contributions and thank you for helping to make 2XD2-\/\+Engine better!}}
